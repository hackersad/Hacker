\subsection*{Aufgabe 7}
Aus einer Lostrommel mit zehn Kugeln, die von 1 bis 10 nummeriert sind, werden 3 Kugeln 
gezogen. Die Zufallsvariable X ist definiert als die größte der drei gezogenen Zahlen. 
Bestimmen Sie die Verteilung von X (durch Angabe der Trägermenge und der 
Wahrscheinlichkeitsfunktion) für den Fall, dass die drei Kugeln 

\begin{enumerate} [label=\alph*)]
\item ohne Zurücklegen gezogen werden.
\vspace{7cm}
\item mit Zurücklegen gezogen werden.
\end{enumerate}
